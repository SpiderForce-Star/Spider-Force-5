# Spider-Force 5

Cockpit-view space combat game.

## Online Play (GitHub Pages)
1. Go to **Settings → Pages**
2. Source: **Deploy from a branch**
3. Branch: **main** / **/ (root)**
4. Save
5. After ~1 minute the game will be live at:
   **https://spiderforce-star.github.io/Spider-Force-5/**

## Controls
- **WASD / Arrow keys**: Pitch & yaw (aim / move view)
- **Hold Shift** while turning: Bank
- **Tap Shift** while turning: Roll
- **Space**: Fire twin blue lasers
- **Enter**: Fire missile (5 per challenge) / Detonate missile for AoE
- **Esc**: Pause

Alien ships attack with red lasers. Use the radar and time your missile detonations carefully!

Built with pure HTML5 Canvas + JavaScript.
